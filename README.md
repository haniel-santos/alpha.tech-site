## Sobre

A **Alpha.Tech** cria sites, sistemas e automações inteligentes para empresas que querem vender mais, atender melhor e economizar tempo. Este repositório contém o código do nosso site institucional — uma landing page moderna e responsiva desenvolvida com as tecnologias mais atuais do ecossistema Next.js.

### Funcionalidades

- **Design escuro e moderno** com gradientes roxo/rosa
- **Animações suaves** com revelação por scroll (`IntersectionObserver`)
- **Totalmente responsivo** — do mobile ao desktop
- **Tema consistente** com componentes reutilizáveis
- **SEO otimizado** com meta tags, Open Graph e JSON-LD
- **Alto desempenho** — site estático gerado com Next.js

## Começando

```bash
# Clone o repositório
git clone https://github.com/AlphaTech-ai/Alpha-Tech-Site.git

# Instale as dependências
npm install

# Inicie o servidor de desenvolvimento
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000) no navegador.

## Scripts

| Comando | Descrição |
|---|---|
| `npm run dev` | Inicia servidor de desenvolvimento (webpack) |
| `npm run build` | Gera build de produção (webpack) |
| `npm run start` | Inicia servidor de produção |
| `npm run lint` | Executa ESLint |
